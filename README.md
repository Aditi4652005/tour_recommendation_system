# 🌍 Tour Recommendation System

A simple and user-friendly **Tour Recommendation System** built using **Python (Flask)** that helps users find travel destinations based on their preferences.

---

## 🚀 Features

* Filter destinations based on:

  * Place Type (Beach, Hill, Adventure, etc.)
  * Budget (Low, Medium, High)
* Display recommended places with images
* Clean and simple user interface
* Fast and dynamic results

---

## 🛠️ Technologies Used

* Python (Flask)
* Pandas
* HTML
* CSS

## ⚙️ How to Run

1. Install required libraries:

pip install flask pandas

2. Run the project:

python app.py

3. Open in browser:

http://127.0.0.1:5000

---

## 💡 How It Works

1. User selects travel preferences
2. Data is sent to Flask backend
3. Dataset is filtered using Pandas
4. Matching destinations are displayed with images
